# Project 3: Spots

## An interactive image-sharing application where users can upload, remove, and engage with photo content

This project is an interactive webpage where users are able to create a profile with their name, an avatar image, and short description of their choosing. Users can upload and remove photos on the grid of their profile and add descriptions or captions to the images post as well. The content is uploaded to a grid on their profile where other users can read and like their content.

## Technologies Used

- HTML5 & CSS3
- Flexbox and Grid layouts
- Google Fonts
- Responsive Design
- Media Queries
- BEM Methodology
- Git/Github
- Figma

## Features

This website uses media queries for responsive design when used on different screen size (e.g desktop, mobile). There are hover effects for the different buttons, a responsive photo grid, as well as handling text overflow using ellipsis.

## Project Goals

This the was the first project where I practiced using media queries as well as hiding text overflow using an ellipsis. In previous projects I have worked with flexbox and grid and in this project I practiced implementing a combination to achieve the layout of the responsive webpage. I also practiced downloading images from figma and adding hover effects to buttons.

## Future Improvements

One of the big challenges I had toward the latter part of the project was adjusting the content on the page to remove horizontal scrolling. I made many adjustment to the content, page and cards blocks and ended up over correcting. One thing that helped me figure out where the overflow was coming from was when I added a red boorder to all elements on the page to highlight where elements started and stopped helped me narrow down what need to be/could be adjusted.
